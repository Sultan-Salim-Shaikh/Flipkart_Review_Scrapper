# Flipkart_Review_Scrapper
Flipkart Review Scrapper is a web based application which is primarly used to scraps the reviews for a particualar products from Ecommerce websites and save the reviews in the csv format that can be further used for Data-Analysis. ReviewScrapper also allows user to limit the reviews based on the Data requirement and scraps the reviews from the multiple pages.
:arrow_right: 
Live : https://review-scrapper-s.herokuapp.com/
